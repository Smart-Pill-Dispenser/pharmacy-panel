# Smart Pill Dispenser · Pharmacy Panel

Pharmacy panel for the Smart Pill Dispenser platform — devices, patients, refills, help & support, logs.

## Run locally

```sh
npm i
npm run dev
```

## Build

```sh
npm run build
```

## Tech

- Vite, TypeScript, React, React Router
- shadcn-ui, Tailwind CSS
- TanStack Query, Recharts
